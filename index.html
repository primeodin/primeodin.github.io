Before the runes.

The early control loop was not a compiler. It was games, typing drills, latency, fog of war, cooldowns, recoil, and the little mercy of a checkpoint placed by someone who understood failure.

StarCraft

RTS

A system can fail before the fight begins. A late depot or missed scout only lets the battle perform the debt already written into the economy.

Warcraft III custom maps

SCRIPT

Broken triggers passed through forums with half an explanation and three new bugs. Code without context becomes folklore too quickly.

Dota Allstars

COOLDOWN

You can have the right ability and still lose because you spent it at the wrong time. Rage is useful only when governed.

Counter-Strike

LATENCY

Angles, economy rounds, recoil discipline, and the truth that milliseconds become architecture before anyone gives you the word architecture.

Mario / Crash / Prince

RECOVERY

The jump is not only the arc. It is the punishment, the retry, the tiny forgiveness window, and the hand learning before the mouth has a theory.

Typing drills

INPUT

Thought has to pass through the hands. If the hands are sloppy, the idea arrives damaged. C made that lesson permanent.

Found artifacts.

Reconstructed textures, not counterfeit proof: notebook pages, source printouts, screenshots, diagrams, and correspondence a person like this might have kept because the object held a rule better than memory did.

runeweaver notebook page, 2003

A torn spiral notebook page with a tiny grammar for let, print, arithmetic, and parentheses. A coffee ring cuts through a stack diagram. In the corner: "do not add loops until errors tell the truth."

freljord engine screenshot, 2004

A 640x480 fixed-function OpenGL test scene: colored boxes, crude winter-blue background, collision normals too bright, and a frame counter ugly enough to be trusted.

PIMC rain printout, 2005

Convergence traces on cheap paper, wrinkled from rain. A bus ticket marks the page. The legend is corrected by hand from "energy" to "estimated energy."

allocator red ink, 2006

A C source printout with red circles around free(ptr): "not done until the table forgets it too." The memory map looks like a small burial ground.

java arena terminal, 2009

TURN 041. score flee=0.72. score attack=0.64. A sticky note says: "when the agent panics, check the map, not the courage."

attention mask photo, 2021

A triangular mask on a whiteboard. B x H x T x T. The bug was not the math. The bug was the shape I lied about.

Winter journals.

Failures and reflections from the reconstructed archive. They are written as memory with caveats, not as recovered public proof.

2006-12-16

If it matters twice, it gets a name. That felt too grand for a cardboard drawer full of switches. Maybe the problem is grandeur without screws in it.

2010-01-08

The best systems I trust are not clever first. They are legible first. Ownership means knowing where the spare cable is because you promised the demo would work.

2014-02-03

The deploy failed for a stupid reason, which is to say, a real reason. We shipped our assumptions to the shell and the shell sent them back.

2018-11-24

Quantum makes me want to talk too much. The math is already strange. It does not need my frost on it.

2022-12-29

Agents are not magic. This is good news. Magic would be another way to avoid responsibility.

Note on memory

Memory is not a database. It is snow with footprints in it. Dates mark the chapter being studied, not a claim of public release.
