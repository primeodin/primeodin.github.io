The road through frost.

Not a portfolio of shiny surfaces. A map of pressure, mistakes, abstractions, and small working machines — from metal-adjacent C to the quantum and language frontiers.

2003–2008

Metal, motion, chalkboard fire

C89, C++, memory, Monte Carlo paths, game loops, and the first diagrams where physics started to look like code with better manners.

2009–2014

Decisions, entropy, lessons, cloud weather

Utility AI, compression experiments, numerical lectures, neural nets, parallel loops, and the first tools for keeping a small kingdom alive on remote machines.

2015–2023

Deep winters, agents, qubits, voice

Deep learning, evolutionary search, reinforcement learning, quantum circuits, resilient services, attention, prompt runes, and a capstone map of the whole path.

What holds.

The archive has a voice, but the voice is not decoration. It is a way to remember why engineering discipline matters.

Small runnable examplesA tiny program that fails clearly teaches more than a cathedral nobody can enter.
Period-aware craftOld code should respect old constraints. Modern code should earn modern abstractions.
Honest mythMetaphor may carry truth, but it must never counterfeit provenance.
